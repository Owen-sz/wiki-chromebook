## UEFI/BIOS Keys

These are the keys (or combonation of keys) used to enter the boot menu or UEFI/BIOS settings on different computers.

| Manufacturer                     | UEFI/BIOS               | Boot Menu                        |
| -------------------------------- | ----------------------- | -------------------------------- |
| Acer                             | F2 or Del               | Esc, F9, F12                     |
| Aorus (Gigabyte)                 | Del, F2                 | F12                              |
| ASRock                           | Del, F2                 | F11                              |
| ASUS                             | F2 or Del               | Esc, F8                          |
| Compaq                           | F10                     | F9                               |
| Dell                             | F12                     | F11                              |
| Dell Tablets                     | Hold Volume down        | Hold Volume up                   |
| ECS                              | Del                     | F7                               |
| EVGA                             | Del                     | F7                               |
| Gateway                          | Del, F1, F2             | F12                              |
| HP                               | ESC, F10                | Esc, F9                          |
| Intel                            | F2                      | F10                              |
| Lenovo (ThinkPad)                | F2, Fn+F2, ThinkVantage | F12, Fn+F12, Enter, ThinkVantage |
| Lenovo Desktops                  | F1                      | F12, Fn+F12                      |
| Lenovo Laptops (Except Thinkpad) | F1+Enter, Novo Button   | F12, Fn+F12, Novo button,        |
| MSI                              | Del                     | F11                              |
| Microsoft (Tablets)              | VolUp+Power             | N/A                              |
| Origin PC                        | F2                      | F2                               |
| Samsung                          | F2                      | F10                              |
| Toshiba                          | F2                      | F12                              |
| Intel Macs                       | n/a                     | Hold Option                      |
| Zotac                            | Del                     | F8                               |
